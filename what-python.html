Что такое Python?

Python - это язык программирования на котором можно писать почти всё что угодно. Это высокоуровневый язык программирования с динамической типизацией, поддерживающий объектно-ориентированный, функциональный и императивный стили программирования. Это язык на котором можно одинаково успешно разрабатывать системные приложения с графическим интерфейсом, утилиты командной строки, научные приложения, игры, приложения для веб и много другое.

У Питона есть мнооожество библиотек, например, Pygame, Django (кстати этот сайт частично написан с его помощью), Flask, Pandas и другие
